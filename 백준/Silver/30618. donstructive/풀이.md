# 풀이
문제는 순열에서 연속부분수열의 합이 최대치가 나오는 순열을 찾는 것입니다. N=4일 때 (1,2,3,4), (2,3,4,1), (3,4,1,2) 등 여러개가 나올 수 있습니다. 각각 연속부분수열이 최대가 되려면 순열의 가운데에 가장 큰 값이 몰려 있으면 됩니다.
### 수학적 접근
길이가 $N$ 인 순열 $i$번째 위치가 포함되는 구간의 개수는 구간의 왼쪽 끝이 $[1, i]$ 오른쪽 끝이 $[i, N]$이므로 총 $i(N-i+1)$개입니다.

계수가 음수인 위가 볼록한 이차함수이므로 i의 위치의 가운데에 큰 값을 놓을수록 순열의 연속부분수열도 커지는 그리디 구조입니다.
### 구현
그렇다면 큰 값을 어떻게 가운데에 몰아줄지가 문제인데 간단하게 앞에서부터 홀수를 오름차순으로, 짝수는 내림차순으로 차례로 배치한 순열을 만들면 됩니다.
## 문제 유형
* 그리디
## 시간 및 공간 복잡도
* 시간복잡도: $\mathcal{O}()$
* 공간복잡도: $\mathcal{O}()$
## 코드
### Python
```python
import sys
def print(*args, sep=" ", end="\n"):
    sys.stdout.write(sep.join(map(str, args)) + end)
def input():
    return sys.stdin.readline().rstrip()

def main():
    n = int(input())
    for i in range(1, n+1, 2): print(i, end=' ')
    for i in range(n-n%2, 0, -2): print(i, end=' ')
if __name__ == '__main__':
    main()
```
