## 풀이
* 시간을 기준으로 최소(left), 최대(right), mid값을 구함
* 시간을 기준으로 이분 탐색을 진행
* mid 시간을 기준으로 몇명이나 가능한지를 검사
* n명 이상이면 시간을 줄이기 위해 right 이동
* 이하이면 시간 초과로 left 이동
* 이걸 반복하면서 최소 시간을 구함
## 자료구조 및 알고리즘
* 이분 탐색
## 문제 유형
* 이분 탐색
## 시간 및 공간 복잡도
* 시간복잡도: $\mathcal{O}(log(max(times) * n) × len(times))$
* 공간복잡도: $\mathcal{O}(1)$
## 코드
```python
def solution(n, times):
    left = min(times)
    result = right = max(times) * n
    
    while left <= right:
        mid = (left + right) // 2
        m = sum(mid // t for t in times)

        if m < n:
            left = mid + 1
        else:
            result = mid
            right = mid - 1
    
    return result
```
